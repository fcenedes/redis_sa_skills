# Redis Product UI Validation

Use `redis-product-ui-demo.html` as a manual validation fixture for the
`redis-product-ui` skill. It is intentionally a static HTML page so agents can
open it directly in a browser and inspect whether the skill guidance produces a
Redis product UI with the expected shell, sidebar, buttons, tables, states,
toasts, and `light2`/`dark2` theme behavior.

This is not product documentation and not a reusable template. Update it only
when the skill guidance changes and the fixture needs to reflect the expected
output.
